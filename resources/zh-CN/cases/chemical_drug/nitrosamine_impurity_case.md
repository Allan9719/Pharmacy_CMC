# 【化学药】发补案例 - 亚硝胺杂质（NDSRI）风险评估不完整（2024）

## 1. 案例背景

**品种类型：** 化学药原料药（含仲胺结构）
**申报阶段：** IND → NDA
**审评机构：** CDE 药学审评部
**发补时间：** 2024 年

CDE 在审评中发现，申请人对原料药中亚硝胺类杂质的风险评估仅覆盖了 NDMA 和 NMBA 两种常见亚硝胺，未对 NDSRI（N-亚硝胺药物相关杂质）进行全面评估。

## 2. 核心问题

### 发补意见摘要

1. **风险评估范围不足**：原料药结构中含有仲胺基团，可能与工艺中引入的亚硝酸盐/硝酸盐在酸性条件下反应生成 NDSRI，但申请人仅评估了 NDMA 和 NMBA
2. **分析方法灵敏度不够**：现有 GC-MS 方法的 LOQ 为 100 ppb，无法达到亚硝胺杂质控制要求（应达 ppb 级）
3. **控制策略缺失**：未针对潜在的 NDSRI 制定合理的控制策略（如工艺优化、中间体质控、成品限度设定）

### CDE 审评思考要点

- 国内外监管机构（FDA、EMA、Health Canada）已对亚硝胺杂质提出全面评估要求
- 亚硝胺杂质来源复杂：起始物料、溶剂（如 DMF 中混入二甲胺）、工艺试剂（亚硝酸钠）、包装系统（亚硝酸盐迁移）
- 企业需基于 ICH M7 评估框架，结合自身工艺进行源头分析

## 3. 整改方案

### 3.1 风险评估补充

```
评估路径：
├── 结构评估
│   ├── 原料药结构中仲胺/叔胺基团识别
│   └── 潜在 NDSRI 结构预测
├── 工艺评估
│   ├── 各步骤中胺类与亚硝酸盐/硝酸盐接触可能性
│   ├── 溶剂回收引入交叉污染风险
│   └── 中间体储存条件（温度、pH、光照）
└── 供应链评估
    ├── 起始物料中亚硝胺残留风险
    └── 辅料/包材中亚硝酸盐迁移
```

### 3.2 分析方法升级

| 项目 | 原方法 | 升级后方法 |
|------|--------|-----------|
| 仪器平台 | GC-MS | LC-MS/MS（三重四极杆） |
| LOQ | 100 ppb | ≤ 1 ppb |
| 检测目标 | NDMA、NMBA | NDMA、NMBA + 结构预测的 NDSRI |
| 定量方式 | 外标法 | 同位素内标法（d6-NDMA 等） |

### 3.3 控制策略

1. **工艺优化**：避免在酸性条件下同时使用仲胺和亚硝酸盐类试剂
2. **中间体控制**：在关键中间体步骤设置亚硝胺检测点
3. **成品放行**：设定各亚硝胺杂质的可接受摄入量（AI），参考 ICH M7 及 EMA 限值
4. **稳定性监测**：在加速和长期稳定性研究中纳入亚硝胺杂质监测

## 4. 审评结论

申请人补充了完整的 NDSRI 风险评估报告，升级分析方法至 LC-MS/MS（LOQ ≤ 1 ppb），并制定了分层控制策略。CDE 认可整改方案，批准进入后续审评流程。

## 5. 避坑建议

1. **不要只关注"常见"亚硝胺**：NDMA/NMBA 只是冰山一角，需结合自身品种结构预测潜在 NDSRI
2. **分析方法灵敏度要提前到位**：ppb 级 LOQ 是基本要求，中国药典尚未收载亚硝胺检测通则，建议参考 USP <1469> 和 EP 2.4.37
3. **从源头控制优于末端检测**：优先优化工艺消除亚硝胺生成条件，而非仅依赖成品检测
4. **关注供应链**：起始物料供应商变更可能引入新的亚硝胺风险，需建立供应商审计机制
5. **保持跟踪监管动态**：EMA 和 FDA 持续更新亚硝胺杂质指南（如 EMA 2024 年更新 Q&A），中国药典预计未来版本将收载相关通则
