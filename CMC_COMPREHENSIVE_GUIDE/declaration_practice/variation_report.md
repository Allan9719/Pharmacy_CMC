# 上市后 CMC 变更申报指南

> 中国药品上市后 CMC 变更的分类、申报流程与实操要点。
>
> 法规依据：《药品上市后变更管理办法（试行）》（2021）、CDE 变更相关指导原则

---

## 1. 变更分类体系

### 1.1 三类变更分类

| 类别 | 定义 | 申报路径 | 审评时限 |
|------|------|---------|---------|
| 微小变更（Class 1） | 对产品质量无明显影响 | 年报报告 | — |
| 中等变更（Class 2） | 对产品质量可能有一定影响 | 备案 | 5-10 个工作日 |
| 重大变更（Class 3） | 对产品质量可能有重大影响 | 补充申请 | 60-200 个工作日 |

### 1.2 常见变更分类示例

#### 原料药变更

| 变更内容 | 分类 | 依据 |
|---------|------|------|
| 分析方法优化（等效或更优） | 微小 | 不影响产品质量 |
| 起始物料供应商变更（标准不变） | 中等 | CDE 变更指南 |
| 合成路线变更 | 重大 | 影响杂质谱 |
| 生产场地变更 | 重大 | 需验证新场地 |
| 质量标准放宽 | 重大 | 直接影响产品可控性 |

#### 制剂变更

| 变更内容 | 分类 | 依据 |
|---------|------|------|
| 辅料供应商变更（同级别） | 中等 | 需可比性数据 |
| 处方变更（辅料用量调整） | 中等/重大 | 依影响程度 |
| 生产工艺变更 | 中等/重大 | 依影响程度 |
| 包装材料变更 | 中等/重大 | 需相容性数据 |
| 有效期延长 | 中等 | 需稳定性数据支持 |

## 2. 申报流程

### 2.1 备案流程（中等变更）

```
变更前评估 → 准备变更资料 → 省级药监局备案
    → 备案完成后实施变更 → 备案后产品监测
```

**备案资料清单：**
1. 变更备案表
2. 变更内容及理由
3. 变更前后对比
4. 变更研究资料（如：对比分析、稳定性数据）
5. 对产品质量的影响评估
6. 变更后产品的检验报告

### 2.2 补充申请流程（重大变更）

```
变更前评估 → 准备补充申请资料 → CDE 受理
    → 技术审评（可能发补）→ 审评通过 → 实施变更
```

**补充申请资料（以场地变更为例）：**
1. 变更内容及理由
2. 新场地 GMP 合规证明
3. 新场地工艺验证报告（≥3 批）
4. 变更前后产品质量可比性研究
5. 稳定性对比数据
6. 分析方法转移/验证报告
7. 对已上市产品的影响评估

## 3. 可比性研究

### 3.1 可比性研究框架

```
变更评估路径：
变更描述 → 影响风险评估
    → 确定需要比较的 CQA
    → 设计可比性方案
    → 执行可比性研究
    → 统计分析
    → 结论：是否可比
```

### 3.2 可比性研究内容

| 研究内容 | 适用场景 | 数据要求 |
|---------|---------|---------|
| 放行检测对比 | 所有变更 | 变更前后 ≥3 批 |
| 扩展表征对比 | 工艺/场地变更 | 全面表征数据 |
| 稳定性对比 | 可能影响稳定性的变更 | 加速 + 长期对比 |
| 溶出曲线对比 | 制剂变更 | f2 因子 ≥50 |
| 生物等效性 | 可能影响 BA/BE 的变更 | BE 试验 |

### 3.3 统计分析方法

- **等价检验（Equivalence Test）**：证明变更前后结果等价
- **容许区间法（Tolerance Interval）**：变更后数据落入历史容许区间内
- **过程能力指数（Cpk）**：变更后过程能力不降低

## 4. 细胞治疗药品变更特殊要求

### 4.1 《细胞治疗药品药学变更研究与评价技术指导原则（试行）》（2026.01.30）

**核心变化：**
- 首次系统性提出细胞治疗药品全生命周期变更要求
- 建立基于风险的变更分级体系
- 明确可比性研究的技术要求

**特殊考虑：**
- 自体细胞治疗：每位患者的"批次"独特性，变更评估需更大样本量
- 病毒载体变更：载体系统的任何变更均为重大变更
- 细胞培养条件变更：需评估对细胞表型、功能的影响

## 5. 变更管理最佳实践

### 5.1 变更控制体系

```
变更控制流程：
1. 变更申请（描述变更内容、原因、预期影响）
2. 变更评估（分类、风险评估、研究方案）
3. 变更审批（QA + 相关部门）
4. 变更执行（按批准方案实施）
5. 变更验证（可比性研究、验证）
6. 变更关闭（评估结果、更新文件）
```

### 5.2 变更记录要求

- 变更编号（唯一追溯码）
- 变更前后描述
- 支持性研究数据
- 审批记录
- 实施日期
- 产品监测数据
