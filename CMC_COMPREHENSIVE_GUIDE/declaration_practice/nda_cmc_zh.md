# 中国 NDA 申报 CMC 实操要点

> 聚焦中国 CDE 框架下 NDA（新药上市申请）阶段 CMC 模块 3 的准备要点。
>
> 法规依据：《药品注册管理办法》（2020）、《药品上市许可申请审评工作程序》

---

## 1. NDA 阶段 CMC 总体要求

### 1.1 核心原则

NDA 阶段 CMC 需证明：
- **质量可控**：商业化生产工艺可重复生产出质量一致的产品
- **工艺稳健**：工艺验证证明在规定参数范围内产品质量可控
- **稳定性充足**：实时稳定性数据支持拟定有效期
- **标准完善**：质量标准基于充分数据，确保每批产品安全有效

### 1.2 IND → NDA 的 CMC 升级路线

| 维度 | IND 阶段 | NDA 阶段需完成 |
|------|---------|---------------|
| 工艺验证 | 初步验证 | ≥3 批商业化规模验证 |
| 质量标准 | 初步标准 | 完整标准 + 统计学依据 |
| 稳定性 | 初步数据 | 实时数据支持有效期 |
| 分析方法 | 部分验证 | 完整 ICH Q2 验证 |
| 杂质 | 主要杂质 | 全面杂质谱 + 基因毒性评估 |
| 包材 | 基本信息 | 完整相容性 + 密封性数据 |
| 对照品 | 工作对照品 | 一级/二级对照品体系 |

## 2. 工艺验证

### 2.1 工艺验证策略

```
工艺验证三阶段（FDA/EMA/CDE 均认可）：
Stage 1：工艺设计（Process Design）
  → 基于开发数据确定 CPP 和 CQA
  → 设计空间建立（DoE）

Stage 2：工艺确认（Process Qualification）
  → 设备确认（IQ/OQ/PQ）
  → 工艺性能确认（PPQ）：≥3 批连续成功
  → 建立商业化控制策略

Stage 3：持续工艺确认（Continued Process Verification）
  → 上市后持续监测 CPP/CQA 趋势
  → 统计过程控制（SPC）
```

### 2.2 验证批次要求

| 项目 | 要求 |
|------|------|
| 批次数 | ≥3 批连续成功 |
| 规模 | 商业化规模（或经验证的缩小比例） |
| 取样 | 关键步骤加强取样 |
| 检测 | 放行标准 + 扩展表征 |
| 结果 | 所有指标符合预设标准 |

### 2.3 验证报告内容

```
工艺验证报告框架：
1. 验证方案（预设标准）
2. 执行摘要
3. 批记录审核
4. 关键工艺参数汇总与趋势分析
5. 关键质量属性汇总与趋势分析
6. 中间产品数据
7. 偏差与评估
8. 结论与建议
```

## 3. 质量标准制定

### 3.1 制定策略

```
质量标准制定数据来源：
├── 非临床批次数据
├── 临床批次数据（I/II/III 期）
├── 工艺验证批次数据
├── 稳定性批次数据
├── 药典标准（ChP/USP/EP/JP）
├── ICH 指导原则（Q3A/B/C/D, Q6A/B）
└── 参比制剂数据（仿制药/生物类似药）
```

### 3.2 标准制定方法

- **统计法**：基于批次数据计算 μ±3σ 或百分位数
- **质量源于设计（QbD）**：基于 CQA 影响评估确定限度
- **药典法**：直接采用药典规定
- **安全性评估法**：基于毒理学数据计算安全限度

## 4. 稳定性研究

### 4.1 NDA 稳定性数据要求

| 条件 | 化学药 | 生物药 |
|------|--------|--------|
| 长期条件 | 25°C/60%RH | 5°C ± 3°C |
| 加速条件 | 40°C/75%RH | 25°C ± 2°C / 60%RH ± 5% |
| 批次数 | ≥3 批 | ≥3 批 |
| 数据要求 | ≥12 个月（长期）+ 6 个月（加速） | ≥12 个月（长期）+ 6 个月（加速） |
| 考察指标 | 外观、含量、有关物质、溶出、水分等 | 外观、纯度、效价、生物活性等 |

### 4.2 有效期制定

```
有效期制定方法：
1. 统计分析法（95% 置信区间，单侧）
2. 外推法（加速数据外推，≤2 倍实际数据时间）
3. 参照已上市同类产品（需论证）
```
