# 中国 IND 申报 CMC 实操要点

> 聚焦中国 CDE 框架下 IND（临床试验申请）阶段 CMC 模块 3 的准备要点。
>
> 法规依据：《药品注册管理办法》（2020）、《药物临床试验质量管理规范》、CDE IND 申报要求

---

## 1. IND 阶段 CMC 总体要求

### 1.1 核心原则

IND 阶段 CMC 旨在证明：
- **安全性**：试验药物在临床试验期间质量可控，受试者安全有保障
- **可追溯性**：原料、辅料、生产过程可追溯
- **一致性**：临床试验用样品与申报资料一致

### 1.2 与 NDA 阶段的区别

| 维度 | IND 阶段 | NDA 阶段 |
|------|---------|---------|
| 生产规模 | 允许实验室/中试规模 | 需商业化 GMP 规模 |
| 工艺验证 | 工艺描述 + 初步验证 | ≥3 批完整工艺验证 |
| 质量标准 | 初步放行标准 | 完整放行标准 + 统计学依据 |
| 稳定性 | 支持 I/II 期临床的初步数据 | 完整有效期稳定性数据 |
| 杂质研究 | 主要杂质鉴定 | 全面杂质谱 + 遗传毒性评估 |
| 分析方法 | 基本验证 | 完整 ICH Q2 验证 |

## 2. CTD 模块 3 撰写框架

### 2.1 化学药 IND（3.2.S + 3.2.P）

```
3.2.S 原料药
├── S.1 一般信息
│   ├── 名称、结构、理化性质
│   └── 分子式、分子量、CAS 号
├── S.2 生产信息
│   ├── 生产商信息
│   ├── 工艺流程图 + 工艺描述
│   ├── 物料控制（起始物料、试剂、溶剂）
│   ├── 过程控制
│   └── 工艺验证（IND 阶段可为初步数据）
├── S.3 特性鉴定
│   ├── 结构确认（NMR、MS、IR、UV、XRD）
│   ├── 理化性质
│   ├── 杂质谱（有机杂质、无机杂质、残留溶剂）
│   └── 生物学活性（如适用）
├── S.4 质量标准
│   ├── 放行标准 + 分析方法
│   ├── 分析方法验证（至少部分验证）
│   └── 批分析数据（≥1 批）
├── S.5 对照品/标准品
│   └── 来源与标定
├── S.6 包装系统
│   └── 内包材信息 + 初步相容性
└── S.7 稳定性
    ├── 稳定性方案
    └── 初步稳定性数据

3.2.P 制剂
├── P.1 处方组成
├── P.2 制剂开发（简要）
├── P.3 生产信息
├── P.4 辅料控制
├── P.5 质量标准
├── P.6 包装系统
└── P.7 稳定性
```

## 3. 高频发补问题与预防

### 3.1 Top 5 IND CMC 发补问题

#### 1. 杂质研究不充分
```
典型发补意见：请补充起始物料引入的杂质研究，并提供基因毒性杂质评估。

预防措施：
- 按 ICH Q3A/Q3B 系统梳理杂质谱
- 按 ICH M7 评估潜在基因毒性杂质
- 起始物料质量标准需包含杂质控制
- 对未知杂质 ≥0.10% 需鉴定
```

#### 2. 稳定性数据不足以支持临床
```
典型发补意见：现有稳定性数据不足以支持拟定临床试验周期。

预防措施：
- IND 申报时至少提供 1-3 个月加速 + 长期数据
- 明确临床试验期间稳定性方案
- 必要时承诺定期提交更新数据
```

#### 3. 分析方法验证不充分
```
典型发补意见：请补充含量测定/有关物质分析方法的验证数据。

预防措施：
- 含量测定：至少完成专属性、线性、准确度、精密度
- 有关物质：至少完成专属性、LOD/LOQ、线性
- 参照药典通则 9101 / ICH Q2
```

#### 4. 生产场地信息不完整
```
典型发补意见：请补充生产场地的 GMP 合规性证明。

预防措施：
- 提供生产许可证 / GMP 证书
- 场地主文件（Site Master File）
- 委托生产需提供质量协议
```

#### 5. 原料药起始物料论证不足
```
典型发补意见：请论证起始物料选择的合理性。

预防措施：
- 按 ICH Q11 论证起始物料选择
- 提供起始物料上游工艺简要描述
- 起始物料质量标准需包含对 CQA 有影响的杂质控制
```

## 4. Pre-IND 沟通会 CMC 准备

### 4.1 建议沟通的 CMC 问题

- 起始物料选择是否可接受
- 质量标准草案是否充分
- 临床试验用样品与商业化工艺的衔接计划
- 特殊产品的药学策略（生物类似药、ADC、ATMP 等）
- 加速审批/突破性治疗的药学支持

### 4.2 沟通会资料准备

```
Pre-IND CMC 沟通资料清单：
├── 产品概述（结构、作用机制、适应症）
├── 生产工艺概述（流程图 + 关键参数）
├── 质量标准草案 + 分析方法概述
├── 初步稳定性数据
├── 杂质谱概述 + 安全性评估计划
├── 拟提出的问题 + 背景说明
└── 支持性数据/文献
```
