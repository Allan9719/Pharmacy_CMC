# 中国药典 2025 版 CMC 影响解读

> 聚焦 2025 年版《中国药典》对 CMC 实操的核心影响，帮助从业者快速理解关键变化并落地应对。
>
> 实施日期：2025 年 10 月 1 日

---

## 1. 总览

### 1.1 核心数据

| 维度 | 数量 | 备注 |
|------|------|------|
| 通用技术要求新增 | 69 个 | 含三部 13 个 + 四部 56 个 |
| 通用技术要求修订 | 133 个 | 含三部 31 个 + 四部 102 个 |
| 指导原则新增 | 33 个 | 含三部 5 个 + 四部 28 个 |
| 指导原则修订 | 17 个 | 均为四部 |
| 制剂通则修订 | 24 个 | 四部 |
| 新增药包材相关指导原则 | 1 个（9624）+ 8 个配套检测方法 | 四部 |

### 1.2 对 CMC 的整体影响

```
药典 2025 版 CMC 影响路径：
├── 分析方法 → 通则 9101 修订，验证要求调整
├── 数据统计 → 通则 9097 新增，统计评估标准化
├── 包装系统 → 通则 4025 新增，跌落性能测试
├── 药包材 → 指导原则 9624 + 8 个检测方法，元素杂质要求
├── 红外光谱法 → 通则修订，影响结构确证
└── 制剂通则 → 24 个制剂通则修订，影响处方/工艺要求
```

## 2. 重点通则解读

### 2.1 通则 9101 — 分析方法验证（修订）

**核心变化：**

| 维度 | 修订前 | 修订后 |
|------|--------|--------|
| 验证要素 | 准确度、精密度、专属性、检测限、定量限、线性、范围、耐用性 | 新增"可报告范围"概念，强化方法生命周期管理 |
| 范围要求 | 覆盖质量标准上下限 | 明确"工作范围"应呈现仪器上最低和最高浓度水平 |
| 专属性 | 要求考察专属性 | 新增"如不具备专属性，可证明选择性"的灵活表述 |
| 统计要求 | 较少涉及 | 增加统计学评价方法，与 ICH Q2(R2) 协调 |

**CMC 实操影响：**
- 已有分析方法需评估是否符合新验证要求
- 新申报项目需按修订后通则执行验证
- 统计学评价要求提升，需加强数据分析能力

**应对建议：**
1. 梳理现有分析方法验证报告，逐项核查是否符合新通则要求
2. 新增验证时补充"可报告范围"研究
3. 加强统计分析（参照通则 9097）

### 2.2 通则 9097 — 数据统计评估（新增）

**核心内容：**

```
通则 9097 主要内容：
├── 描述性统计
│   ├── 集中趋势（均值、中位数）
│   ├── 离散程度（标准差、极差）
│   └── 统计学图形（直方图、箱线图）
├── 统计推断
│   ├── 置信区间
│   └── 假设检验
└── 方法学应用
    ├── 分析方法验证中的统计评价
    ├── 稳定性数据评价
    └── 工艺能力评估
```

**CMC 实操影响：**
- 质量标准制定需基于统计学依据（如 μ±3σ）
- 稳定性数据评价需采用统计方法（回归分析、置信区间）
- 工艺验证数据分析需运用统计工具

### 2.3 通则 4025 — 塑料包装系统抗跌落性能测定法（新增）

**核心内容：**

| 参数 | 要求 |
|------|------|
| 适用范围 | 注射液用塑料瓶/袋、口服液体药用塑料瓶、外用液体药用塑料瓶等 |
| 跌落高度 | 1000mm-2000mm（常见 1.0m、1.2m） |
| 跌落角度 | 0°（瓶底）、30°、90°（侧面） |
| 释放方式 | 气动夹紧释放，避免人为误差 |
| 评价标准 | 跌落后包装完整性（无泄漏、无破裂） |

**CMC 实操影响：**
- 新增塑料包装制剂需补充跌落性能测试
- 运输验证需考虑跌落测试数据支持
- 包材供应商需提供符合通则 4025 的检测数据

### 2.4 药包材相关新增

**指导原则 9624 + 8 个配套检测方法：**
- 塑料包装系统质量控制新要求
- 元素杂质检测（参照 ICH Q3D）
- 新增剥离强度、拉伸性能等检测方法修订

## 3. 制剂通则修订影响

### 3.1 修订的 24 个制剂通则

制剂通则修订数量最多，主要涉及：
- 注射剂通则修订（强化无菌保证、配伍稳定性要求）
- 口服固体制剂通则修订（溶出度、含量均匀度要求调整）
- 外用制剂通则修订（新增质量控制项目）

### 3.2 CMC 申报影响

| 制剂类型 | 关键修订 | 申报影响 |
|---------|---------|---------|
| 注射剂 | 无菌保证、配伍稳定性 | 需补充配伍稳定性研究 |
| 口服固体 | 溶出度要求调整 | 需更新溶出曲线分析方法 |
| 外用制剂 | 新增质控项目 | 需补充检测方法和验证 |

## 4. 与 ICH 的协调进展

### 4.1 与 ICH Q2(R2) 的协调

通则 9101 修订参照了 ICH Q2(R2) 的最新要求：
- 新增"可报告范围"概念
- 强化统计学评价
- 引入方法生命周期管理理念

### 4.2 与 ICH Q14 的关系

- 中国尚未正式采纳 ICH Q14（分析方法开发）
- 通则 9101 的修订已部分融入 Q14 的理念
- 建议：分析方法开发可参照 Q14 的框架，同时满足 9101 要求

## 5. 企业应对时间线

```
药典 2025 应对时间线：
2025.10 前：
  ├── 梳理现有产品涉及的修订通则
  ├── 评估现有分析方法验证的合规差距
  └── 制定整改计划

2025.10-2026.03：
  ├── 补充不符合新通则的验证数据
  ├── 更新质量标准（如涉及）
  └── 培训相关人员（统计方法、新检测方法）

2026.03+：
  ├── 持续监测新通则执行中的问题
  ├── 跟踪药典勘误和补遗
  └── 更新企业内控标准
```
