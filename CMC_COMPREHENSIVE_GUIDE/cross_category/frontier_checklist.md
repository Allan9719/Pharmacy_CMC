# 前沿疗法 CMC 合规自查清单

> 适用于 ATMP / mRNA / ADC / 寡核苷酸等前沿疗法药物 CMC 申报前的系统性自查。
>
> 使用方法：逐项检查，确保在 IND/NDA 提交前全部符合。

---

## 1. 通用合规清单

### 1.1 生产场地合规

| 序号 | 检查项 | 是/否 | 备注 |
|------|--------|------|------|
| 1 | 生产许可证 / GMP 证书有效 | | |
| 2 | 场地主文件（SMF）已更新 | | |
| 3 | 委托生产协议 + 质量协议签署（如适用） | | |
| 4 | 多场地职责划分明确 | | |
| 5 | 设备确认（IQ/OQ/PQ）完成 | | |
| 6 | 清洁验证完成（含交叉污染评估） | | |

### 1.2 质量管理体系

| 序号 | 检查项 | 是/否 | 备注 |
|------|--------|------|------|
| 1 | QA/QC 独立设置 | | |
| 2 | 批记录体系完整 | | |
| 3 | 偏差/CAPA 体系运行 | | |
| 4 | 变更控制体系运行 | | |
| 5 | 供应商审计体系运行 | | |
| 6 | 年度产品质量回顾（APQR）制度 | | |

### 1.3 分析方法

| 序号 | 检查项 | 是/否 | 备注 |
|------|--------|------|------|
| 1 | 放行标准完整（鉴别、纯度、含量、安全指标） | | |
| 2 | 分析方法按 ICH Q2 / 药典 9101 验证 | | |
| 3 | 对照品体系建立（一级/二级） | | |
| 4 | 分析方法转移完成（多场地时） | | |
| 5 | OOS/OOT 处理程序建立 | | |

### 2.4 质量标准
