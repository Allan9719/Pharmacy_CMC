# ADC 药物 CDE 发补与整改指南

> 聚焦中国 CDE 审评框架下 ADC（抗体偶联药物）的药学（CMC）发补高频问题与整改策略。
> 📖 基于公开审评报道及行业实践整理，不构成 CDE 官方审评标准。
>
> 核心法规：《抗体偶联药物药学研究与评价技术指导原则》（CDE, 2024.02.08）

---

## 1. 监管框架

### 1.1 核心法规依据

| 法规/指导原则 | 发布时间 | 核心内容 |
|--------------|----------|---------|
| 《抗体偶联药物药学研究与评价技术指导原则》 | 2024.02.08 | 中国首个 ADC 药学专项指南，九章内容系统覆盖 ADC 全链条 |
| 《抗体偶联药物临床试验药理学指导原则》 | 2024.09 | ADC 临床药理学研究要求 |
| ICH Q6B | — | 生物制品质量标准 |
| ICH Q5E | — | 可比性研究 |
| 中国药典 2025 版 | 2025.10 | 通则 9101 分析方法验证等 |

### 1.2 ADC 结构与 CMC 挑战

```
ADC 分子结构：
┌──────────┐    Linker    ┌──────────┐
│  抗体     │────────────│  小分子   │
│ (Antibody)│  (连接子)   │  毒素     │
│  ~150 kDa │             │(Payload) │
└──────────┘              └──────────┘

CMC 核心挑战：
├── 三组分的分别质控 + 偶联后质控
├── DAR（药物抗体比）分布的精确控制与表征
├── 游离毒素/连接子的安全管理
├── 异质性强（位置异构、DAR 异构）
└── 多场地协作（抗体生产 → 毒素合成 → 偶联 → 灌装）
```

## 2. 三组分的分别质控

### 2.1 裸抗（Antibody）

| 质控维度 | 关键项目 | 放行标准参考 |
|---------|---------|------------|
| 鉴别 | 肽图、等电点 | 与参比品一致 |
| 纯度 | SEC-HPLC、CE-SDS | 纯度 ≥95% |
| 效价 | 结合活性（ELISA/SPR） | ≥ 参比品的 80% |
| 电荷异构 | CEX-HPLC / iCIEF | 酸性/碱性/主峰比例 |
| 糖基化 | HILIC / CE | 糖型分布 |
| 残留 Protein A | ELISA | ≤10 ppm |
| 内毒素 | 鲎试剂 | ≤5 EU/mg |
| 微生物限度 | 薄膜过滤 | 符合规定 |

**ADC 特殊要求：**
- 裸抗需满足偶联反应的特定位点要求（如链间二硫键还原度）
- 游离巯基含量需精确控制
- 需建立偶联前裸抗的"偶联适用性"评价标准

### 2.2 小分子毒素（Payload/Linker）

| 质控维度 | 关键项目 | 法规依据 |
|---------|---------|---------|
| 结构确认 | NMR、MS、IR、UV | ICH Q6A |
| 纯度 | HPLC（有关物质） | 药典标准 |
| 手性纯度 | 手性 HPLC / 旋光度 | 依化合物要求 |
| 残留溶剂 | GC | ICH Q3C |
| 遗传毒性杂质 | 按 ICH M7 评估 | ICH M7 |
| 重金属 | ICP-MS | ICH Q3D |
| 稳定性 | 影响因素 + 加速 + 长期 | ICH Q1A |

**毒素安全管理：**
- 高活性化合物（OEB 4-5 级）需在隔离器/负压手套箱内操作
- 职业暴露限值（OEL）评估
- 废弃物按危险化学废弃物处理

### 2.3 连接子（Linker）

| 连接子类型 | 代表 | 释放机制 | CMC 关注点 |
|-----------|------|---------|-----------|
| 可裂解连接子 | MC-Val-Cit-PABC | 酶切释放（组织蛋白酶 B） | 酶切速率、血浆稳定性 |
| 不可裂解连接子 | SMCC | 抗体降解后释放 | 代谢产物谱 |
| pH 敏感连接子 | Hydrazone | 酸性内涵体释放 | pH 依赖性释放动力学 |

## 3. 偶联工艺控制

### 3.1 偶联工艺流程

```
裸抗还原/氧化（打开链间二硫键）
    → 与毒素-连接子偶联反应
    → 淬灭反应
    → 纯化（去除游离毒素、副产物）
    → 配方/超滤
    → 除菌过滤
    → 分装
```

### 3.2 关键工艺参数

| CPP | 控制要求 | 影响 |
|-----|---------|------|
| 还原剂（如 TCEP/DTT）当量 | 精确控制摩尔比 | 反应巯基数 → DAR |
| 毒素-连接子当量 | 优化确定 | DAR、偶联效率 |
| 反应 pH | 精确控制（如 pH 6.5-7.5） | 偶联位点选择性 |
| 反应温度 | 严格控制 | 偶联速率、副反应 |
| 反应时间 | 优化确定 | DAR、偶联完全度 |
| 淬灭条件 | 精确控制 | 终止反应一致性 |

### 3.3 DAR 控制策略

```
DAR（Drug-to-Antibody Ratio）控制：
├── 目标 DAR：通常 2-4（依产品设计）
├── DAR 分布：HIC-HPLC 或 LC-MS 表征
├── 批间一致性：RSD ≤ 10%
└── 高/低 DAR 物种需分别定量与控制
```

## 4. ADC 成品质控

### 4.1 放行标准（参考）

| 质控项目 | 方法 | 放行标准 |
|---------|------|---------|
| 鉴别 | 肽图 + 质谱 | 与参比品一致 |
| DAR（平均） | HIC-HPLC / UV/UV | 目标值 ± 0.5 |
| DAR 分布 | HIC-HPLC | D0-D8 各组分比例符合规定（适用于 IgG1 链间二硫键偶联；其他偶联策略范围不同） |
| 纯度（SEC） | SEC-HPLC | 单体 ≥95% |
| 纯度（CE-SDS） | 还原/非还原 CE-SDS | 符合规定 |
| 游离毒素 | RP-HPLC | ≤3% |
| 游离抗体 | SEC-HPLC | ≤5% |
| 效价（结合活性） | ELISA/SPR | ≥ 参比品的 80% |
| 效价（细胞毒性） | 细胞杀伤实验 | ≥ 参比品的 80% |
| 电荷异构 | CEX / iCIEF | 符合规定 |
| 糖基化 | HILIC | 符合规定 |
| 内毒素 | 鲎试剂 | ≤5 EU/mg |
| 无菌 | 薄膜过滤 | 无菌 |
| 不溶性微粒 | 光阻法 | 符合药典 |

### 4.2 特有分析技术

| 分析技术 | 用途 | 法规参考 |
|---------|------|---------|
| HIC-HPLC | DAR 分布分析 | CDE ADC 指南 |
| LC-MS（完整蛋白/亚基） | DAR 精确测定 + 位点分析 | CDE ADC 指南 |
| DSC / DSF | 热稳定性 | 表征 |
| SEC-MALS | 分子量分布 | 表征 |
| CLND（化学发光氮检测） | DAR（基于氮含量） | 替代方法 |
| 亲水相互作用色谱 | 糖基化分析 | ICH Q6B |

## 5. 稳定性研究

### 5.1 稳定性指标

| 类别 | 关键指标 | 变化关注点 |
|------|---------|-----------|
| 物理稳定性 | SEC 纯度、不溶性微粒 | 聚集体增加 |
| 化学稳定性 | DAR 变化、游离毒素 | 连接子断裂、毒素脱落 |
| 生物学活性 | 结合活性、细胞毒性 | 效价下降 |
| 外观 | 颜色、澄清度 | 变色、浑浊 |

### 5.2 特殊考虑

- **游离毒素增长趋势**：需在稳定性考察中追踪，建立可接受标准
- **DAR 分布变化**：不同 DAR 物种比例在储存期间可能漂移
- **光敏感性**：部分毒素（如 PBD）对光敏感，需避光储存

## 6. CDE 发补高频问题与整改

### 6.1 2024-2026 年 CDE 高频发补问题

#### 问题 1：DAR 分布控制策略不充分
```
发补意见：请提供 DAR 分布的批间一致性数据，并说明 DAR 分布对产品安全性和有效性的影响。

整改方案：
1. 汇总 ≥10 批次 DAR 分布数据（HIC-HPLC），统计分析批间一致性
2. 提供不同 DAR 物种（D0-D8）的体内外活性比较数据
3. 结合非临床 PK/PD 数据，论证 DAR 分布的可接受范围
4. 建立 DAR 分布的统计过程控制（SPC）图
```

#### 问题 2：游离毒素限度制定依据不足
```
发补意见：请提供游离毒素限度的制定依据，包括非临床毒理学数据支持。

整改方案：
1. 提供游离毒素的非临床毒理学数据（MTD、NOAEL）
2. 基于非临床数据计算安全窗口
3. 结合多批次生产数据，采用统计方法制定限度（如 μ+3σ）
4. 提供稳定性考察中游离毒素增长趋势数据
```

#### 问题 3：偶联工艺验证不充分
```
发补意见：请补充商业化规模偶联工艺验证数据，特别是 DAR 可控性验证。

整改方案：
1. 补充 ≥3 批商业规模验证数据
2. 验证 CPP 范围的最差条件（worst case）
3. 证明在工艺参数范围内 DAR 可控且批间一致
4. 提供工艺稳健性研究数据（Design of Experiments）
```

#### 问题 4：毒素杂质谱研究不充分
```
发补意见：请完善小分子毒素的有关物质研究，并提供遗传毒性杂质评估。

整改方案：
1. 补充强制降解试验，鉴定降解产物
2. 按 ICH M7 评估遗传毒性杂质风险
3. 建立已知杂质的定量方法并验证
4. 制定杂质限度（参照 ICH Q3A/Q3B）
```

#### 问题 5：分析方法验证不完整
```
发补意见：请按 ICH Q2 / 药典通则 9101 完善以下分析方法验证：
  (a) DAR 测定方法（HIC-HPLC）
  (b) 游离毒素定量方法
  (c) 效价测定方法

整改方案：
1. 逐一按 ICH Q2 要求完成验证报告
2. DAR 方法：准确度（加标回收）、精密度（6 批重复）、专属性、线性
3. 游离毒素：LOD/LOQ、准确度、精密度、线性
4. 效价方法：精密度、中间精密度、线性、范围
```

### 6.2 发补整改通用策略

```
整改流程：
1. 逐条分析发补意见 → 分类（数据补充/方法完善/论证加强）
2. 制定整改计划 → 责任人 + 时间线
3. 执行整改 → 实验/研究/文献调研
4. 撰写回复 → 逐条回应 + 附件证据
5. 内部质控 → QA 审核后提交
```

## 7. 供应链与多场地管理

### 7.1 ADC 典型供应链

```
场地 A：抗体原液生产（ mammalian cell culture → 纯化 → 制备）
          ↓ （冷链运输，质量协议约束）
场地 B：毒素-连接子合成（化学合成 → 纯化 → 质检）
          ↓ （特殊运输，安全管理）
场地 C：偶联生产（偶联反应 → 纯化 → 配方 → 分装）
          ↓
场地 D：成品包装与放行
```

### 7.2 多场地管理要点

- **质量协议**：明确各场地职责、放行标准、变更通知要求
- **运输验证**：中间产品的冷链运输验证
- **分析方法转移**：确保各场地分析结果可比
- **变更协调**：任一场地变更需评估对最终产品的影响
