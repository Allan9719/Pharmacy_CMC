# mRNA 药物本土质控要点

> 聚焦中国 NMPA/CDE 监管框架下 mRNA 药物的 CMC 关键要求。
>
> 法规依据：《预防用 mRNA 疫苗药学研究技术指导原则（试行）》（CDE, 2026.01.27）、中国药典 2025 版

---

## 1. 监管背景

### 1.1 中国 mRNA 药物法规进展

| 时间 | 法规/事件 | CMC 影响 |
|------|----------|---------|
| 2020-2023 | 新冠 mRNA 疫苗紧急使用 | EUA 路径下的药学要求 |
| 2024 | 多款 mRNA 候选药物进入 IND | 个性化肿瘤疫苗、传染病疫苗 |
| 2026.01.27 | 《预防用 mRNA 疫苗药学研究技术指导原则（试行）》 | 首个系统性 mRNA 药学指导原则，覆盖序列设计至质量控制全流程 |

### 1.2 产品分类

| 类型 | 代表方向 | CMC 特点 |
|------|---------|---------|
| 预防性 mRNA 疫苗 | 新冠、流感、RSV | 大规模生产，LNP 递送 |
| 治疗性肿瘤疫苗 | 个性化新抗原 mRNA | 个性化生产，快速周转 |
| 蛋白替代疗法 | 罕见病酶替代 | 高表达量需求，长效化 |
| 基因编辑递送 | mRNA 编码 Cas9 | 编辑效率 + 安全性双重质控 |

## 2. 原液（Drug Substance）质控

### 2.1 DNA 模板控制

```
DNA 模板质控要求：
├── 序列确认（Sanger/NGS 全序列测定）
├── 纯度（A260/A280 比值 1.8-2.0）
├── 超螺旋比例（≥90%）
├── 残留 RNA 污染
├── 宿主菌 DNA 残留
├── 内毒素（≤10 EU/mg）
└── 无菌检测
```

### 2.2 mRNA 原液质控

| 质控项目 | 方法 | 放行标准（参考） | 法规依据 |
|---------|------|-----------------|---------|
| 鉴别（序列确认） | RT-PCR / 测序 | 与设计序列一致 | CDE 指导原则 |
| 完整性（完整性百分比） | 毛细管电泳（CE） / HPLC | ≥80%（或产品规定） | CDE 指导原则 |
| 5' 加帽率 | LC-MS / 免疫法 | ≥90% | CDE 指导原则 |
| 3' Poly(A) 尾长度 | LC-MS / PCR | 符合规定范围 | CDE 指导原则 |
| 纯度 | RP-HPLC / IP-RP-HPLC | ≥90% | 药典 9101 |
| dsRNA 残留 | ELISA / 免疫印迹 | ≤ 规定限度 | 安全性要求 |
| 残留 DNA 模板 | qPCR | ≤10 ng/剂量 | WHO/CDE |
| 残留宿主蛋白 | BCA / ELISA | ≤ 规定限度 | CDE 指导原则 |
| 残留酶（T7 RNA 聚合酶等） | ELISA | ≤ 规定限度 | 工艺相关杂质 |
| 残留有机溶剂 | GC / HPLC | 符合 ICH Q3C | ICH Q3C |
| 内毒素 | 鲎试剂 | ≤5 EU/mL | 药典通则 1143 |
| 无菌 | 薄膜过滤法 | 无菌 | 药典通则 1101 |

### 2.3 体外转录（IVT）工艺控制

**关键工艺参数：**

| 参数 | 控制要求 | 影响 |
|------|---------|------|
| NTP 浓度与比例 | 优化确定，严格控制 | 产量、完整性 |
| T7 RNA 聚合酶用量 | 按摩尔比精确投料 | 转录效率 |
| 反应温度 | 37°C ± 2°C | 酶活性、产物质量 |
| 反应时间 | 优化确定 | 产量 vs 副产物 |
| Mg²⁺ 浓度 | 精确控制 | 酶催化效率 |
| 加帽策略 | 共转录 vs 酶法加帽 | 加帽率、成本 |

### 2.4 纯化工艺

```
典型纯化流程：
IVT 产物 → DNase 消化（去除模板 DNA）
    → 纤维素色谱 / 疏水色谱（去除 dsRNA、短片段）
    → 反相色谱 / 离子交换（纯化目标 mRNA）
    → 切向流过滤（TFF，缓冲液置换 + 浓缩）
    → 除菌过滤（0.22 μm）→ 冻存或直接制剂
```

## 3. LNP 制剂质控

### 3.1 脂质成分控制

| 脂质组分 | 功能 | 质控要求 |
|---------|------|---------|
| 可电离脂质（ALC-0315 / SM-102 等） | 内涵体逃逸 | 纯度 ≥99%、杂质谱、降解产物 |
| PEG 脂质（ALC-0159 / DMG-PEG2000） | 粒径控制、空间稳定 | PEG 分子量分布、纯度 |
| 胆固醇 | 膜稳定性 | 来源（合成/天然）、纯度 |
| DSPC / 辅助磷脂 | 膜结构 | 纯度、氧化降解产物 |

**可电离脂质特有关注点：**
- 降解产物谱（氧化、水解产物）
- 遗传毒性评估
- 新脂质需单独申报安全性数据

### 3.2 LNP 关键质量属性

| 属性 | 方法 | 控制范围（参考） |
|------|------|-----------------|
| 粒径（Z-average） | 动态光散射（DLS） | 60-100 nm |
| PDI（多分散指数） | DLS | ≤0.25 |
| Zeta 电位 | 电泳光散射 | -10 ~ +10 mV（中性范围） |
| 包封率 | RiboGreen / 荧光法 | ≥90% |
| mRNA 浓度 | UV260 / RiboGreen | 符合规定 |
| 脂质/mRNA 比例 | HPLC + UV260 | 符合设计比例 |
| 包裹效率 | 酶消化法 | ≥85% |

### 3.3 制剂处方

```
典型 mRNA-LNP 制剂处方：
├── mRNA-LNP 核心颗粒（含水合缓冲液）
├── 缓冲体系（如 Tris/蔗糖缓冲液）
├── 冻干保护剂（如适用：蔗糖、海藻糖）
├── pH 调节剂
└── 注射用水

关键处方参数：
- pH：7.0-8.0（可电离脂质最佳包封 pH）
- 渗透压：280-320 mOsm/kg
- 蔗糖浓度：5-10%（冻存保护）
```

## 4. 稳定性研究

### 4.1 稳定性方案设计

```
mRNA-LNP 稳定性研究矩阵：
├── 长期稳定性
│   ├── -80°C / -20°C（冻存态）
│   └── 2-8°C（液态，如适用）
├── 加速稳定性
│   ├── 25°C ± 2°C
│   └── 40°C ± 2°C
├── 冻融稳定性
│   └── 至少 3 次冻融循环
├── 使用中稳定性
│   └── 复溶/稀释后室温放置
└── 运输稳定性
    ├── 冷链运输模拟
    └── 温度偏移试验
```

### 4.2 关键稳定性指标

| 指标类别 | 检测项目 | 变化可接受标准 |
|---------|---------|--------------|
| 物理稳定性 | 粒径、PDI、外观 | 粒径增长 ≤20%，PDI ≤0.30 |
| 化学稳定性 | mRNA 完整性、加帽率、Poly(A) | 完整性下降 ≤10% |
| 制剂稳定性 | 包封率、脂质降解 | 包封率 ≥85% |
| 安全性 | 无菌、内毒素 | 符合放行标准 |
| 功能性 | 体外表达活性 | ≥ 放行标准的 80% |

### 4.3 冷链管理

```
mRNA-LNP 冷链示例：
生产 → -80°C 冻存 → 干冰运输（-60°C ~ -80°C）
→ 接收后在 -80°C 或 -20°C 储存
→ 使用前 2-8°C 解冻 → 室温稀释 → 即时使用

温度偏离处理：
- 记录偏离时间与温度
- 评估产品质量影响（稳定性数据支持）
- 超出可接受范围 → 隔离处理
```

## 5. 分析方法验证要求

### 5.1 按 ICH Q2(R2) / 药典通则 9101 执行

| 方法类型 | 验证要求 |
|---------|---------|
| 鉴别（测序、RT-PCR） | 专属性 |
| 含量测定（HPLC、UV） | 准确度、精密度、线性、范围、专属性 |
| 杂质定量（dsRNA、残留 DNA） | 准确度、精密度、检测限、定量限、线性 |
| 粒径/DLS | 准确度、精密度 |
| 包封率 | 准确度、精密度、线性 |
| 生物学活性 | 精密度、中间精密度 |

### 5.2 特殊考虑

- **加帽率分析**：LC-MS 方法需验证质量准确性、分辨率
- **完整性分析**：CE 方法需验证片段分离能力
- **dsRNA 检测**：免疫法需验证特异性（避免 ssRNA 交叉反应）

3.2.S.1 一般信息 — mRNA 序列、长度、修饰（如假尿苷修饰）

## 7. 常见发补问题

### 7.1 CDE 高频发补问题

1. **mRNA 完整性数据不充分** → 补充多批次 CE 数据 + 降解动力学研究
2. **dsRNA 残留限度依据不足** → 提供非临床安全性数据 + 临床相关文献
3. **LNP 表征不完整** → 补充结构表征（Cryo-EM、SAXS）+ 批间一致性数据
4. **加帽率检测方法未验证** → 按 ICH Q2 完成方法验证
5. **稳定性数据不足以支持有效期** → 补充实时稳定性数据至拟定有效期
6. **脂质杂质谱研究不充分** → 补充降解产物鉴定 + 遗传毒性评估
7. **冷链运输验证不完整** → 补充温度偏移 + 冻融循环数据
8. **残留 DNA 模板限度论证不足** → 提供残留序列分析 + 风险评估
