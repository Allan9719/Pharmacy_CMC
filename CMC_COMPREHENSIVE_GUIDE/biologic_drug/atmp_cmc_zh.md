# 中国 ATMP 药物 CMC 申报实操指南

> 本文档聚焦中国 NMPA/CDE 监管框架下，前沿疗法药物（ATMP）的药学（CMC）申报核心要求与实操要点。
>
> 适用范围：细胞治疗药物、基因治疗药物、组织工程产品

---

## 1. 监管框架

### 1.1 核心法规依据

| 法规/指导原则 | 发布机构 | 发布时间 | 核心内容 |

| 《基因治疗产品药学研究与评价技术指导原则（征求意见稿）》 | CDE | 2023 | 基因治疗药学评价框架 |
| 《细胞治疗药品药学变更研究与评价技术指导原则（试行）》 | CDE | 2026.01.30 | 首个细胞治疗全生命周期变更专项指南 |
| 《体内基因治疗产品药学研究与评价技术指导原则（试行）》 | CDE | 2024 | 体内基因治疗药学要求 |

### 1.2 ATMP 分类与对应要求

| ATMP 类型 | 代表产品 | CMC 核心挑战 | 监管侧重 |
|-----------|----------|-------------|---------|
| CAR-T | 阿基仑赛、瑞基奥仑赛 | 自体生产追溯、病毒载体质控、批间一致性 | 个性化生产合规 |
| TCR-T | NY-ESO-1 TCR-T | 靶点特异性验证、TCR 序列确认 | 安全性评价 |
| 干细胞治疗 | MSC、ESC 来源产品 | 多能性维持、分化方向控制、致瘤性评估 | 成药性论证 |
| 基因治疗（体外） | β-地贫基因治疗 | 载体整合效率、脱靶分析、表达持续性 | 长期安全性 |
| 基因治疗（体内） | AAV 基因治疗 | 载体纯度、空壳率、免疫原性 | 剂量与安全性 |
| mRNA 疫苗/药物 | 新冠 mRNA 疫苗 | LNP 表征、序列完整性、冷链稳定性 | 质量可控性 |

## 2. 原料与起始材料控制

### 2.1 细胞来源控制

#### 自体来源
```
供体筛查清单：
├── 身份确认（身份证 + 病历号双验证）
├── 传染病筛查（HIV、HBV、HCV、梅毒、HTLV）
├── 既往治疗史评估（化疗/放疗对细胞质量影响）
├── 知情同意书签署
└── 采样时机与条件记录
```

**关键质控点：**
- 采样至生产启动的时限控制（通常 ≤72h）
- 运输过程温度监控（2-8°C 或室温，依产品要求）
- 单采血/肿瘤组织质量评估（细胞活力 ≥70%）

#### 异体来源
- 需额外建立供体库管理系统
- HLA 配型记录（若适用）
- 供体筛查标准参照《人体细胞组织器官质量安全管理规范》

### 2.2 病毒载体控制

| 载体类型 | 生产系统 | 关键质控项目 |
|---------|---------|------------|
| 慢病毒载体 | 293T 四质粒瞬时转染 | 滴度、感染性、复制型慢病毒（RCL）、残留宿主 DNA |
| 逆转录病毒载体 | 稳定包装细胞系 | 滴度、RCL、包装细胞残留、内毒素 |
| AAV 载体 | 293T/昆虫细胞-Sf9 | 基因组滴度、空壳率、衣壳蛋白比例、残留核酸 |
| 腺病毒载体 | 293 细胞 | 感染性滴度、复制型腺病毒（RCA）、残留宿主 DNA |

### 2.3 关键试剂与辅料

**高风险试剂清单：**
- 磁珠分选试剂（CD3/CD28 等）— 残留量控制
- 细胞因子（IL-2、IL-7、IL-15 等）— 来源与纯度
- 培养基 — 无血清/成分明确要求
- 转染试剂 — 残留与安全性评估
- 冻存保护剂（DMSO）— 终产品残留限度

## 3. 生产工艺控制

### 3.1 工艺流程（以 CAR-T 为例）

```
单采血采集 → T 细胞分选/富集 → T 细胞活化
    → 慢病毒转导 → 扩增培养 → 收获与洗涤
    → 配方与分装 → 冻存 → 质检放行
```

### 3.2 关键工艺参数（CPP）

| 工艺步骤 | CPP | 控制范围 | 监测方法 |
|---------|-----|---------|---------|
| T 细胞活化 | CD3/CD28 磁珠比例 | 1:1 ~ 3:1 | 流式细胞术 |
| 慢病毒转导 | MOI（感染复数） | 1 ~ 10 | qPCR |
| 扩增培养 | 细胞密度 | 0.5-2×10⁶/mL | 细胞计数仪 |
| 扩增培养 | 培养天数 | 7-14 天 | 在线监测 |
| 收获洗涤 | 磁珠残留去除 | < 检测限 | 磁珠检测 |
| 冻存 | 降温速率 | -1°C/min | 程序降温仪 |

### 3.3 工艺验证要求

**批次要求：**
- 临床阶段：至少 2-3 批工艺验证批次
- 商业化阶段：至少 3 批连续成功批次
- 验证参数范围应覆盖最差条件（worst case）

**过程控制（IPC）：**
```
每个关键步骤需设置 IPC 取样点：
├── 活化后：细胞活力、CD3+ 比例
├── 转导后：转导效率（CAR+ 比例）
├── 扩增过程：细胞计数、活力、CAR+ 维持率
├── 收获后：总细胞数、CAR+ 绝对数、杂质去除
└── 冻存后：复苏活力、功能检测
```

### 3.4 污染控制策略

- **无菌控制**：封闭式生产系统优先，A级环境下开放操作
- **支原体控制**：每批检测，培养法 + PCR 法双重确认
- **内毒素控制**：动态鲎试剂法，放行标准 ≤5 EU/kg/hr
- **交叉污染**：专场专用或充分清洁验证

## 4. 质量标准与放行

### 4.1 CAR-T 产品放行标准（参考）

| 检测项目 | 方法 | 放行标准 | 参考依据 |
|---------|------|---------|---------|
| 外观 | 目视 | 符合规定（澄清/微乳白色） | 中国药典 |
| 鉴别（CAR 表达） | 流式细胞术 | CAR+ ≥ 规定比例 | CDE 指导原则 |
| 细胞活力 | 流式/台盼蓝 | ≥70%（或产品规定） | 药典通则 |
| 活细胞数 | 计数仪 | 符合规定范围 | 工艺验证 |
| CAR+ 细胞数 | 流式 + 计数 | ≥ 最低有效剂量 | 临床数据 |
| 残留磁珠 | 磁性检测/显微镜 | < 检测限 | 安全性要求 |
| 内毒素 | 鲎试剂 | ≤5 EU/kg/hr | 药典通则 1143 |
| 无菌 | 薄膜过滤法 | 无菌 | 药典通则 1101 |
| 支原体 | 培养 + PCR | 阴性 | 药典通则 3301 |
| 复制型慢病毒（RCL） | PCR/共培养 | 阴性 | CDE 指导原则 |
| 残留宿主 DNA | qPCR | ≤10 ng/剂量 | WHO/CDE |
| 残留 BSA | ELISA | ≤50 ng/10⁶ 细胞 | 安全性评估 |

### 4.2 基因治疗产品特有质控

| 产品类型 | 特有质控项目 |
|---------|------------|
| AAV 产品 | 空壳率（≤30%）、衣壳蛋白比例（VP1:VP2:VP3=1:1:10）、基因组完整性 |
| 体外基因治疗 | 基因编辑效率、脱靶分析、载体拷贝数（VCN）、插入位点分析 |
| 体内基因治疗 | 载体分布、表达持续性、免疫原性评估 |

## 5. 稳定性研究

### 5.1 稳定性研究设计

```
ATMP 产品稳定性研究框架：
├── 原液（Drug Substance）
│   ├── 短期稳定性：冻存条件、复苏后时效
│   └── 运输稳定性：冷链模拟、温度偏移试验
├── 成品（Drug Product）
│   ├── 冻存稳定性：-150°C / -80°C / 液氮气相
│   ├── 复溶/复苏后稳定性：2-8°C / 室温
│   └── 使用中稳定性：输液袋中稳定性
└── 中间产品
    └── 各工艺中间体暂存条件与时限验证
```

### 5.2 考察指标

- **常规指标**：外观、细胞活力、活细胞数、CAR/转基因表达率
- **功能指标**：体外杀伤活性、细胞因子分泌能力
- **安全指标**：无菌、内毒素、支原体（特定时间点）

### 5.3 运输验证

| 验证项目 | 要求 |
|---------|------|
| 运输容器验证 | 保温时长验证（≥ 规定运输时间 1.5 倍） |
| 温度监控 | 全程温度记录（连续数据记录仪） |
| 温度偏移 | 模拟温度偏移场景，评估产品质量影响 |
| 机械振动 | 运输模拟振动试验 |
| 目的地放行 | 到达后需重新检测关键指标 |

## 6. CTD 模块 3 撰写要点

### 6.1 3.2.S 原液部分

```
3.2.S.1 一般信息
  → 细胞来源描述、遗传修饰方式、载体信息

3.2.S.2 生产信息
  → 工艺流程图（含 CPP/CMA 标注）
  → 工艺描述（含规模、设备、参数范围）
  → 过程控制
  → 工艺验证方案与结果

3.2.S.3 特性鉴定
  → 鉴别与纯度
  → 物理化学与生物学特性
  → 杂质分析（工艺相关 + 产品相关）

3.2.S.4 质量标准
  → 放行标准 + 依据
  → 分析方法验证

3.2.S.5 对照品/标准品
  → 参比品建立与标定

3.2.S.6 包装系统
  → 冻存袋/瓶相容性、密封完整性

3.2.S.7 稳定性
  → 稳定性方案 + 数据
```

### 6.2 3.2.P 制剂部分

- ATMP 产品通常 S 和 P 为同一阶段，制剂部分需描述最终配方
- 冻存保护剂配方与浓度论证
- 分装容器系统（冻存袋/西林瓶）相容性数据

## 7. IND 阶段 vs NDA 阶段 CMC 要求对比

| 维度 | IND 阶段 | NDA 阶段 |
|------|---------|---------|
| 生产场地 | 允许临床/研发场地 | 需商业化 GMP 场地 |
| 工艺验证 | 2-3 批验证 | ≥3 批连续商业规模验证 |
| 质量标准 | 初步放行标准 | 完整放行标准 + 统计学依据 |
| 稳定性 | 支持 I/II 期临床的稳定性数据 | 支持有效期的完整稳定性数据 |
| 分析方法 | 基本验证 | 完整验证（按 ICH Q2 / 药典 9101） |
| 杂质研究 | 已知杂质鉴定 | 全面杂质谱分析 + 遗传毒性评估 |
| 变更管理 | 记录变更 | 变更前后可比性研究 |

## 8. 变更管理（2026 新规要点）

### 8.1 《细胞治疗药品药学变更研究与评价技术指导原则》核心要求

**变更分类：**

| 变更类别 | 示例 | 申报路径 |
|---------|------|---------|
| 重大变更 | 生产场地变更、载体系统变更、关键质量标准变更 | 补充申请 |
| 中等变更 | 工艺参数范围调整、培养基供应商变更 | 备案 |
| 微小变更 | 分析方法优化（等效或更优）、标签变更 | 年报报告 |

### 8.2 变更可比性研究

**必须进行可比性研究的场景：**
- 生产场地变更
- 生产工艺变更
- 规模变更
- 关键原材料变更
- 质量标准变更

**可比性研究内容：**
1. 质量对比分析（放行指标 + 扩展表征）
2. 工艺对比（收率、工艺参数可比性）
3. 稳定性对比
4. 非临床功能性对比（如适用）

## 9. 常见发补问题与应对

### 9.1 高频发补问题 Top 10

1. **载体拷贝数（VCN）限度依据不充分** → 需提供非临床安全性数据 + 临床数据支持
2. **杂质研究不完整** → 需完善工艺相关杂质（磁珠残留、DNA 残留等）研究
3. **稳定性数据不足以支持有效期** → 需补充实时稳定性数据
4. **工艺验证批次不足** → 需补充商业化规模验证批次
5. **质量标准制定依据不明确** → 需提供临床试验批次数据 + 统计分析
6. **分析方法未验证或验证不充分** → 按 ICH Q2 / 药典 9101 完善验证
7. **运输验证不完整** → 需补充运输模拟 + 温度偏移数据
8. **供体筛查标准不明确** → 需完善供体纳入/排除标准
9. **交叉污染防控措施不足** → 需加强清洁验证 + 专场论证
10. **变更管理不完善** → 需建立变更控制体系 + 可比性研究

### 9.2 应对策略

```
发补应对流程：
1. 逐条分析发补意见，归类（数据补充型 / 方法学改进型 / 机制说明型）
2. 制定整改计划，明确时间线
3. 补充研究数据 / 完善分析方法 / 提供法规依据
4. 撰写回复文档，逐条回应
5. 内部审核后提交
```
