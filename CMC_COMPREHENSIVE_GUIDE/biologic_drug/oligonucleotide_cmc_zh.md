# 化学合成寡核苷酸药物 CMC 申报要点

> 聚焦中国 CDE 监管框架下化学合成寡核苷酸药物（ASO、siRNA、核酸适配体）的药学（CMC）核心要求。
> 📖 基于 CDE《化学合成寡核苷酸药物（创新药）药学研究技术指导原则（试行）》（2026.02.24）及行业实践整理。

---

## 1. 背景与法规框架

### 1.1 药物类型

| 类型 | 全称 | 作用机制 | 代表药物 |
|------|------|---------|---------|
| ASO | 反义寡核苷酸 | 与靶 mRNA 互补配对，调控基因表达 | Spinraza (nusinersen) |
| siRNA | 小干扰 RNA | 双链结构，通过 RISC 介导 mRNA 降解 | Onpattro (patisiran), Amvuttra (vutrisiran) |
| Aptamer | 核酸适配体 | 三维结构结合靶蛋白 | Macugen (pegaptanib) |

> **适用范围：** 本文档覆盖化学合成寡核苷酸药物（创新药），不包含 mRNA 等生物合成药物。

### 1.2 核心法规

| 法规名称 | 发布机构 | 日期 |
|---------|---------|------|
| 《化学合成寡核苷酸药物（创新药）药学研究技术指导原则（试行）》 | CDE | 2026.02.24 |
| ICH Q8/Q9/Q10/Q11 | ICH | 通用 |
| 《药品注册管理办法》 | NMPA | 2020 |

## 2. 原料药（Drug Substance）CMC 要点

### 2.1 起始物料控制

寡核苷酸固相合成的起始物料包括：

| 物料类别 | 具体内容 | 控制要点 |
|---------|---------|---------|
| 核苷亚磷酰胺单体 | A、T(U)、G、C 四种碱基单体 | 化学纯度 ≥ 99%、杂质谱、水分 |
| 固相载体 | CPG（可控微孔玻璃珠） | 载量、孔径、初始核苷酸连接效率 |
| 修饰单体 | 2'-OMe、2'-F、2'-MOE、LNA 等 | 纯度、立体化学、杂质 |
| 硫代试剂 | 如 Beaucage 试剂（PS 修饰用） | 纯度、残留 |
| 溶剂/试剂 | 乙腈、氧化剂、封端试剂 | 残留溶剂控制（ICH Q3C） |

### 2.2 生产工艺

```
固相合成工艺流程（典型）：
├── 固相载体装载
├── 合成循环（3'-5' 方向，每个循环含）：
│   ├── 脱保护（Detritylation） — 移除 DMT 保护基
│   ├── 偶联（Coupling） — 核苷亚磷酰胺连接
│   ├── 氧化/硫代（Oxidation/Sulfurization） — 形成磷酸二酯/硫代磷酸酯键
│   └── 封端（Capping） — 封闭未反应的 5'-OH
├── 裂解与脱保护（Ammonia treatment）
├── 纯化（RP-HPLC / IEX-HPLC / 切向流过滤）
├── 脱盐 / 超滤
└── 冻干 → 原料药
```

**关键工艺参数：**
- 偶联效率（目标 > 99%/步，20 聚体总收率约 80%+）
- 硫代率（PS 修饰时，目标 > 98%）
- 粗品纯度（影响纯化难度和收率）

### 2.3 杂质控制

寡核苷酸杂质种类复杂，需多原理分析方法：

| 杂质类别 | 来源 | 分析方法 |
|---------|------|---------|
| 截短序列（N-X） | 偶联不完全/未封端 | IEX-HPLC, LC-MS |
| 长序列（N+X） | 脱保护不完全导致双偶联 | IEX-HPLC, LC-MS |
| 硫代磷酸酯键异构体（Sp/Rp） | 硫代反应立体选择性 | RP-HPLC, ³¹P NMR |
| 脱氨基产物 | 碱基修饰降解 | LC-MS |
| 残留保护基 | 脱保护不完全 | LC-MS |
| 工艺相关杂质 | 溶剂、试剂残留 | GC, ICP-MS |

### 2.4 结构确证

| 表征项目 | 方法 |
|---------|------|
| 分子量 | ESI-MS / MALDI-TOF MS |
| 序列确认 | LC-MS/MS 酶解测序 |
| 硫代位点/比例 | ³¹P NMR, LC-MS |
| 立体化学（如适用） | ³¹P NMR |
| 二级结构（siRNA 双链） | CD（圆二色谱）、UV 熔解温度（Tm） |
| 纯度 | IEX-HPLC, RP-HPLC, CGE（毛细管凝胶电泳） |

## 3. 制剂（Drug Product）CMC 要点

### 3.1 递送系统

| 递送技术 | 适用类型 | 关键质控参数 |
|---------|---------|-------------|
| LNP（脂质纳米粒） | siRNA（如 Onpattro） | 粒径（50-100 nm）、PDI、包封率、Zeta 电位 |
| GalNAc 偶联 | ASO/siRNA（如 Givlaari） | 偶联效率、GalNAc 密度、靶向活性 |
| 裸露注射 | ASO（如 Spinraza） | 鞘内注射无菌/内毒素控制 |

### 3.2 制剂质量控制

| 检测项目 | 方法 | 备注 |
|---------|------|------|
| 含量 | UV（A260） | 摩尔消光系数法 |
| 纯度/杂质 | IEX-HPLC / RP-HPLC | 降解产物监测 |
| 无菌 | 薄膜过滤法 | 注射剂必检 |
| 内毒素 | 鲎试剂法（动态比浊/比色） | 注射剂 |
| pH / 渗透压 | pH 计 / 渗透压计 | |
| 可见异物 | 灯检法 | |
| 包封率（LNP） | 荧光染料排阻法 / RiboGreen | LNP 制剂 |
| 粒径/Zeta（LNP） | DLS | LNP 制剂 |

## 4. 稳定性研究

```
稳定性方案要点：
├── 原料药
│   ├── 长期：-20°C ± 5°C（冻干粉）
│   ├── 加速：5°C ± 3°C / 25°C ± 2°C
│   └── 考察指标：纯度、杂质、水分、外观
├── 制剂
│   ├── 长期：按拟定贮藏条件
│   ├── 加速：按 ICH Q1A
│   ├── 使用中稳定性（多剂量包装）
│   └── 考察指标：含量、纯度、降解产物、无菌、内毒素
└── 特殊考虑
    ├── siRNA 双链在冻融/稀释条件下的链分离
    ├── LNP 制剂的脂质氧化/水解
    └── GalNAc 偶联物的连接子稳定性
```

## 6. CDE 高频发补问题（基于公开信息整理）

| # | 发补问题 | 要点 |
|---|---------|------|
| 1 | 起始物料控制不足 | 核苷亚磷酰胺单体的杂质谱未充分表征 |
| 2 | 杂质界定不完整 | 截短序列、硫代异构体未按 ICH Q3A 原则分类和界定 |
| 3 | 分析方法验证不充分 | IEX-HPLC 方法的专属性、耐用性未充分验证 |
| 4 | 工艺验证数据不足 | 商业化规模的合成偶联效率、纯化收率数据不充分 |
| 5 | 递送系统质控缺失 | LNP 粒径分布、包封率、脂质降解产物控制不完整 |
| 6 | 稳定性降解途径研究不充分 | 未鉴定主要降解产物或未建立降解趋势 |
| 7 | 结构确证不完整 | 硫代位点/比例、立体化学未充分表征 |

## 7. IND vs NDA 申报差异

| 维度 | IND 阶段 | NDA 阶段 |
|------|---------|---------|
| 工艺 | 中试规模，描述完整即可 | 商业化规模，工艺验证 ≥3 批 |
| 杂质 | 初步杂质谱，主要杂质界定 | 完整杂质谱，所有 ≥ 报告阈值杂质界定 |
| 方法验证 | 基本验证 | 完整验证（ICH Q2 / 药典通则 9101） |
| 稳定性 | ≥ 3 月加速 + ≥ 6 月长期 | ≥ 6 月加速 + ≥ 12 月长期 |
| 结构确证 | 分子量 + 序列 + 基本纯度 | 完整表征（含立体化学、硫代比例） |
| 递送系统 | 初步质控 | 完整质控 + 体外释放 + 体内相关性 |
