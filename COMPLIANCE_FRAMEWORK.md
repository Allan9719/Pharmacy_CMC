## 二、实时更新机制

### 2.1 自动化监控（零代码）

| 监控平台                        | 工具                                | 提醒方式  | 时效     |
| ------------------------------- | ----------------------------------- | --------- | -------- |
| CDE 指导原则/共性问题/eCTD/通告 | Feedly RSS + ChangeDetection/盯站侠 | 微信/邮件 | ≤5 分钟  |
| NMPA 政策法规→药品监管          | 网页监控 + 官方公众号               | 微信      | ≤10 分钟 |
| 药典委公告/通则修订             | RSS 订阅                            | 邮件      | 实时     |

**兜底措施**：
- 置顶微信公众号：CDE、NMPA、药监局药品审评中心
- 每周固定巡检（10 分钟）：手动检查 CDE 指导原则页 + 共性问题页

### 2.2 分级预警

| 级别       | 触发条件                                                         | 处理时限 |
| ---------- | ---------------------------------------------------------------- | -------- |
| 🟥 一级紧急 | CDE 指导原则发布/修订、杂质紧急管控、GMP 重大更新、eCTD 规则变更 | 12 小时  |
| 🟧 二级常规 | CDE 共性答疑、药典通则修订、MAH/变更管理补充要求                 | 3 天     |
| 🟩 三级无关 | 医疗器械/中药/化妆品法规                                         | 忽略     |

## 三、三重校验门（强制执行）

### 第一重：唯一权威源校验
- ❌ 禁止：自媒体解读、行业论坛帖文、非官方课件
- ✅ 必须：粘贴 CDE/NMPA/药典委官方原文链接作为每条更新的依据

### 第二重：条款对标校验
- 逐字逐句对照官方原文，禁止自行改写或概括
- 示例：修改「工艺验证」→ 打开 CDE《药品生产工艺验证技术指导原则》→ 直接提取条款

### 第三重：术语锁死校验
- 强制使用 CDE 官方术语表，不自定义翻译
- 全文档术语必须一致，官方修订术语时同步全文更新

## 四、标准化更新 SOP（6 步流水线）

```
Step 1: 自动捕获 → 分级标记
        收到提醒 → 打开原文 → 标记优先级 → 录入更新台账

Step 2: 评估影响 → 定位章节
        判断影响范围 → 精准定位指南章节

Step 3: 对标修改 → 原文引用
        用官方条款替换旧内容 → 标注法规溯源（机构+日期+文件名）

Step 4: 双人复核
        个人维护：间隔 2 小时二次核对
        团队维护：1 人修改、1 人审核

Step 5: 更新版本号 + 日志
        在指南首页标注更新时间/内容/合规依据
        在 UPDATE_TRACKER.md 新增记录

Step 6: 发布生效
        同步更新 .md + PDF 版本
```

## 六、高频误区规避清单

| #   | 误区                             | 正确做法                    |
| --- | -------------------------------- | --------------------------- |
| 1   | 直接套用 FDA/EMA 要求            | 用中国 GMP/CDE 指导原则替代 |
| 2   | 忽略 MAH、关联审评等中国特色制度 | 专章覆盖                    |
| 3   | 引用过期药典（2020 版）          | 统一引用 2025 版            |
| 4   | 术语不统一（DMF 翻译混乱）       | 锁定 CDE 官方译法           |
| 5   | 化药/生物药/细胞治疗 CMC 混用    | 分章节、分类别说明          |
