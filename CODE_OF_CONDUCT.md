# 行为准则

## 一、核心准则

1. **尊重包容**：尊重不同背景、不同岗位从业者的观点，禁止人身攻击
2. **专业务实**：聚焦 CMC 实操问题讨论，禁止无关内容
3. **合规诚信**：所有内容需基于中国药品监管法规，禁止传播虚假信息
4. **保密原则**：禁止泄露企业未公开的敏感信息

## 二、禁止行为

1. 发布违反中国法律法规的内容
2. 发布歧视、侮辱、攻击性言论
3. 发布与 CMC 无关的广告、推广、引流内容
4. 篡改 / 删除他人合法贡献的内容
5. 泄露企业 / 个人隐私、商业秘密

## 三、违规处理

1. 首次违规：警告并删除违规内容
2. 多次违规：禁止参与仓库协作
3. 严重违规：保留证据并上报相关平台

## 四、争议解决

1. 内容准确性争议：以 NMPA / CDE / 药典委官方文件为最终依据
2. 协作流程争议：联系核心维护者沟通
3. 违规处理争议：可提交申诉，核心维护者 7 个工作日内回复
